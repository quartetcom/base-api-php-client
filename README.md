PHP client library for accessing [BASE API](https://developers.thebase.in/).

## Requirements

* PHP 5.4+

## Getting started

Just add these dependency into your `composer.json`:

```json
{
    "require": {
        "quartetcom/base-api-php-client": "1.0.*@dev",
        "cakephp/utility": "3.0.*@beta"
    }
}
```
